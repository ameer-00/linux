## shell
>**shell :**  Shell is a program that takes keyboard commands, interprets them, then translates them into requests the operating system can understand and execute via system calls.
>>note : the layer between the shell and the (OS) is **system call** or **kernel interface**
>
>**bash :** Bash stands for “Bourne Again SHell”, and it is an enhanced and free replacement for the original Unix shell “sh”.
>>Most (linux) distributions come bundled with the (Shell program) like Ubuntu,Linux Mint,Kali Linux etc... .
>> In windows there is two shells (CMD , PowerShell).
> 
>**Terminal emulator :** User interface GUI , you need another program called a terminal emulator or (konsole) to interact with the  shell . 
>
