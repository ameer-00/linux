cd (Change directory) : to move around the linux filesystem .
    cd . (current directory) : Represents the directory you are currently in .

ls (List Directories)  : will list the directories and files in your current directory .
    ls -a (all) : you can view all files and hidden files .
    ls -l (long) : show detailed information starting from the left , file permissions, number of links, owner name, owner group, file size, timestamp of last modification, and the file or directory name.
    ls -r (reverse) : lest files and directories in reverse alphabetical order . 
    ls -al || ls -lar  :  al >> all and long    &&     lar >> all , long and reverse   .

    touch -d (date and time) : can ste a files timestamp to a specific date and time ,   ( "2023-01-01 12:30:00" file name ) .

cp : To copy a file you specify the source file and the destination directiory or path  cp [source] [destination]
    [ source ] : The file or folder you want to copy .
    [ destination ]: The place where you want to put the copy . 
    cp -r(recursive) :To copy a directory and all of its content , including subdirectories . 
    cp -i(interactive) :By default, cp will overwrite a file at the destination if it has the same name. To prevent accidental data loss, use the -i (interactive) flag, which prompts for confirmation before overwriting.
    cp -p :  is particularly useful for backups or when migrating files where preserving timestamps is critical.

mv (move) : it serves two primary purposes >> 
    renaming files or directories :  for renaming 

    moving them to a different location : to move items from one location to another . 
        Note: ' mv ' does not leave a copy of the original file , unlike 'cp'  which copies and leaves the original.
        -t : which allows to specify the target directory first . 
        -i (interactive): it will prompt you for confirmation before overwriting any existing file.
        -b (backup) : If you intend to overwrite a file but want to keep the old version, this option creates a backup of the destination file. The backup is typically renamed with a tilde (~) suffix.
        -v (verbose):  This option makes the mv command print out what it is doing, showing each file being moved or renamed.

Using Wildcards or globbing patterns for buld copying : 
        *  :  Matches any sequence of characters.
        ?  :  Matches any single character.
        []  :  Matches any one of the characters enclosed in the brackets.

mkdir (make Directory) : create a directory .

rm (remove) : deleting files and directories . 
    -f (force)  : To bypass these safety prompts and remove files unconditionally, you can use the force option.
    -i (interactive) : flag prompts you for confirmation before deleting each file, helping to prevent accidental removal
    -r (recursive) : delete a directory and all of its contents.
    rmdir : command will only succeed if the directory is completely empty .


find : to search a file in specific directory , [path] [expression] .
    [path] : directory the search in . 
    [expression] : what i looking for . 
    -name : option allows you to specify the exact name of the file you want to find . 
    -type : specify the type of item you are searching for . 
    -type d : to find a directory instead of a file .
